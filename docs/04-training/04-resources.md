## Cheat Sheets

- `php artisan list` — every Artisan command, grouped.
- `php artisan about` — versions of everything Laravel knows about.
- `php artisan route:list` — every route and its handler.
- Pint config: `pint.json` — code style rules in force.
- Eloquent query builder reference: <https://laravel.com/docs/queries>.
